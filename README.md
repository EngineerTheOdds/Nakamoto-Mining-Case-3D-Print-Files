# Nakamoto Mining Case — 3D Print Files

STL files for printing the Nakamoto Mining case enclosure and its accessories.

> **Design credit:** All designs are by **Nakamoto Mining** — https://nakamotomining.com/en-us
> This repository only hosts the print files for convenience. Please support the original designer.

## Parts

| File | Description |
| --- | --- |
| `MIDDLE PIECE (Hashboard Enclosure).stl` | Main hashboard enclosure / middle body |
| `Control Board Cover .stl` | Cover for the control board |
| `Terminal Cover.stl` | Cover for the terminal / connections |
| `Handle.stl` | Carry handle |
| `INTAKE (6inch Inline Fan).stl` | Air intake for a 6-inch inline fan |
| `EXHAUST (6inch Inline Fan).stl` | Exhaust for a 6-inch inline fan |
| `INTAKE (Dual 120mm Fans).stl` | Air intake for dual 120 mm fans |
| `EXHAUST (Dual 120mm Fans).stl` | Exhaust for dual 120 mm fans |
| `EXHAUST (Fanless Honeycomb) .stl` | Passive honeycomb exhaust (fanless) |

## Cooling options

Mix and match the intake/exhaust pieces to suit your setup:

- **6-inch inline fan** — pair `INTAKE (6inch Inline Fan)` with `EXHAUST (6inch Inline Fan)`.
- **Dual 120 mm fans** — pair `INTAKE (Dual 120mm Fans)` with `EXHAUST (Dual 120mm Fans)`.
- **Fanless** — use `EXHAUST (Fanless Honeycomb)` for passive airflow.
